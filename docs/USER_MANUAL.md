## 시작하기

### 로그인

1. 브라우저에서 인사관리 시스템 URL 접속
2. 로그인 페이지에서 이메일과 비밀번호 입력
3. **로그인** 버튼 클릭

**참고**: 처음 사용하는 경우 관리자에게 계정 생성 요청이 필요합니다.

### 회원가입

1. 로그인 페이지에서 **회원가입** 링크 클릭
2. 이메일, 비밀번호, 비밀번호 확인 입력
3. **회원가입** 버튼 클릭
4. 관리자에게 역할(admin, hr, viewer) 부여 요청

### 역할별 권한

- **관리자 (admin)**: 모든 기능 사용 가능
- **인사담당자 (hr)**: 직원 관리, 파일 관리, 엑셀 연동 가능
- **조회자 (viewer)**: 조회만 가능 (수정/삭제 불가)

## 직원 관리

### 직원 목록 조회

1. 메인 페이지에서 전체 직원 목록 확인
2. 각 직원 카드에 다음 정보 표시:
   - 프로필 이미지
   - 이름
   - 부서
   - 직급
   - 입사일

### 직원 검색

1. 상단 검색창에 검색어 입력
2. 이름, 부서, 직급으로 자동 검색
3. 실시간으로 검색 결과 업데이트

**팁**: 검색어를 입력하면 즉시 필터링됩니다.

### 필터링

1. 상단 필터 영역에서 필터 선택:
   - **부서**: 드롭다운에서 부서 선택
   - **직급**: 드롭다운에서 직급 선택
   - **입사일 범위**: 시작일과 종료일 선택
   - **재직 상태**: 재직/휴직/퇴사 선택

2. 여러 필터를 조합하여 사용 가능

3. 필터 초기화: **필터 초기화** 버튼 클릭

### 정렬

1. 상단 정렬 옵션에서 정렬 기준 선택:
   - 이름순
   - 입사일순
   - 급여순
   - 부서순

2. 오름차순/내림차순 토글 버튼으로 변경

### 필터 프리셋 저장

1. 필터 설정 후 **필터 저장** 버튼 클릭
2. 프리셋 이름 입력
3. 저장된 프리셋은 드롭다운에서 선택하여 즉시 적용 가능
4. 프리셋 삭제: 프리셋 옆 **삭제** 버튼 클릭

### 새 직원 등록

1. 메인 페이지에서 **새 직원 등록** 버튼 클릭
2. 직원 정보 입력:
   - **필수 항목**: 사원번호, 이름, 이메일, 부서, 직급, 입사일
   - **선택 항목**: 전화번호, 생일, 주소, 급여, 자격증, 경력 등
   - **계약 정보**: 계약 시작일, 종료일, 갱신일
3. 프로필 이미지 업로드 (선택)
4. **저장** 버튼 클릭

**주의사항**:
- 사원번호와 이메일은 중복될 수 없습니다
- 중복 체크는 실시간으로 수행됩니다

### 직원 정보 수정

1. 직원 카드 클릭하여 상세 정보 열기
2. **수정** 버튼 클릭
3. 정보 수정
4. **저장** 버튼 클릭

**권한**: admin, hr 역할만 수정 가능

### 직원 삭제

1. 직원 상세 정보에서 **삭제** 버튼 클릭
2. 확인 대화상자에서 **삭제** 확인

**권한**: admin 역할만 삭제 가능

**주의**: 삭제된 직원은 복구할 수 없습니다.

### 직원 상세 정보

직원 카드를 클릭하면 다음 정보를 확인할 수 있습니다:

- 기본 정보 (이름, 사원번호, 부서, 직급 등)
- 연락처 정보
- 입사 정보
- 급여 정보
- 자격증 정보
- 경력 정보
- 계약 정보
- 파일 관리 링크

### 인사관리카드 출력

1. 직원 상세 정보에서 **인사관리카드 출력** 버튼 클릭
2. 인쇄 대화상자에서 **인쇄** 또는 **PDF로 저장** 선택

## 대시보드

### 대시보드 접근

1. 네비게이션 바에서 **대시보드** 클릭
2. 또는 `/dashboard` URL로 직접 접근

### 통계 정보

대시보드에서 다음 통계를 확인할 수 있습니다:

- **전체 직원 수**: 현재 재직 중인 직원 수
- **부서별 인원**: 각 부서별 직원 수
- **직급별 분포**: 각 직급별 직원 수
- **입사/퇴사 추이**: 최근 30일간 입사/퇴사 추이

### 차트 및 그래프

1. **부서별 인원 파이 차트**: 부서별 인원 분포 시각화
2. **입사 추이 라인 차트**: 시간에 따른 입사 추이
3. **급여 분포 히스토그램**: 급여 분포 시각화

### 리포트 다운로드

1. 대시보드 상단 **리포트 다운로드** 버튼 클릭
2. 형식 선택:
   - **CSV**: 엑셀에서 열 수 있는 형식
   - **JSON**: 프로그래밍에서 사용 가능한 형식
   - **텍스트**: 읽기 쉬운 텍스트 형식
3. 파일이 자동으로 다운로드됩니다

## 알림 시스템

### 알림 확인

1. 네비게이션 바의 알림 아이콘 확인
2. 읽지 않은 알림 개수가 빨간 배지로 표시됩니다
3. 알림 아이콘 클릭하여 알림 목록 페이지로 이동

### 알림 종류

시스템은 다음 알림을 제공합니다:

1. **자격증 만료 알림**
   - 30일 전 알림 (우선순위: 낮음)
   - 7일 전 알림 (우선순위: 중간)
   - 만료된 자격증 (우선순위: 높음)

2. **계약 갱신 알림**
   - 30일 전 알림 (우선순위: 낮음)
   - 7일 전 알림 (우선순위: 중간)
   - 만료된 계약 (우선순위: 높음)

3. **생일 알림**
   - 오늘 생일 (우선순위: 중간)
   - 7일 후 생일 (우선순위: 낮음)

### 알림 관리

1. **읽음 처리**: 알림 항목 클릭 또는 **읽음 처리** 버튼 클릭
2. **삭제**: 알림 항목의 **삭제** 버튼 클릭
3. **모두 읽음 처리**: 상단 **모두 읽음 처리** 버튼 클릭

### 알림 자동 업데이트

- 알림 개수는 30초마다 자동으로 업데이트됩니다
- 다른 탭에서 알림을 읽으면 현재 탭에도 반영됩니다

## 파일 관리

### 파일 관리 페이지 접근

1. 직원 상세 정보에서 **파일 관리** 링크 클릭
2. 또는 `/employees/[직원ID]/files` URL로 직접 접근

### 파일 업로드

1. **파일 업로드** 버튼 클릭
2. 파일 선택 대화상자에서 파일 선택
3. 파일 타입 선택:
   - **문서**: 일반 문서
   - **이력서**: 이력서 (버전 관리 가능)
   - **계약서**: 계약서 (만료일 관리 가능)
   - **기타**: 기타 파일
4. 설명 입력 (선택)
5. 계약서인 경우 만료일 입력
6. **업로드** 버튼 클릭

**제한사항**:
- 파일 크기: 최대 10MB
- 지원 형식: PDF, 이미지, Word 문서 등

### 파일 다운로드

1. 파일 목록에서 다운로드할 파일 찾기
2. **다운로드** 버튼 클릭
3. 파일이 자동으로 다운로드됩니다

### 파일 삭제

1. 파일 목록에서 삭제할 파일 찾기
2. **삭제** 버튼 클릭
3. 확인 대화상자에서 **삭제** 확인

**주의**: 삭제된 파일은 복구할 수 없습니다.

### 이력서 버전 관리

1. 이력서 파일 업로드 시 자동으로 버전 번호가 할당됩니다
2. 최신 버전은 **최신 버전** 배지로 표시됩니다
3. 다른 버전을 최신 버전으로 설정:
   - 해당 이력서의 **최신 버전으로 설정** 버튼 클릭

### 계약서 만료일 관리

1. 계약서 업로드 시 만료일 입력
2. 만료일 수정:
   - 계약서의 **만료일 수정** 버튼 클릭
   - 새 만료일 입력
   - **저장** 버튼 클릭

### 파일 필터링

파일 목록 상단에서 파일 타입별로 필터링 가능:
- 전체
- 문서
- 이력서
- 계약서
- 기타

## 엑셀 연동

### 엑셀 임포트 (일괄 등록)

1. 네비게이션 바에서 **엑셀 임포트** 클릭
2. **템플릿 다운로드** 버튼 클릭하여 템플릿 파일 다운로드
3. 템플릿 파일에 직원 정보 입력
4. **파일 선택** 버튼 클릭하여 엑셀 파일 선택
5. **임포트** 버튼 클릭
6. 임포트 결과 확인:
   - 성공한 직원 수
   - 실패한 직원 수
   - 오류 상세 정보
   - 임포트된 직원 목록

**주의사항**:
- 사원번호와 이메일은 중복될 수 없습니다
- 필수 항목이 누락된 경우 임포트가 실패합니다
- 오류가 있는 행은 건너뛰고 나머지는 임포트됩니다

### 엑셀 템플릿 형식

템플릿 파일에는 다음 컬럼이 포함됩니다:

- 사원번호 (필수)
- 이름 (필수)
- 이메일 (필수)
- 부서 (필수)
- 직급 (필수)
- 입사일 (필수)
- 전화번호
- 생일
- 주소
- 급여
- 기타 필드

**팁**: 템플릿 파일의 첫 번째 행은 헤더입니다. 두 번째 행부터 데이터를 입력하세요.

### 엑셀 익스포트 (다운로드)

1. 메인 페이지에서 **엑셀 다운로드** 버튼 클릭
2. 현재 필터/검색 조건에 맞는 모든 직원 데이터가 엑셀 파일로 다운로드됩니다

**권한**: admin, hr 역할만 다운로드 가능

## FAQ

### 일반 질문

**Q: 로그인할 수 없습니다.**
A: 다음을 확인하세요:
- 이메일과 비밀번호가 올바른지 확인
- 계정이 생성되었는지 관리자에게 확인
- 브라우저 캐시 삭제 후 다시 시도

**Q: 직원을 삭제할 수 없습니다.**
A: 삭제 권한은 admin 역할만 있습니다. 관리자에게 권한 확인을 요청하세요.

**Q: 파일을 업로드할 수 없습니다.**
A: 다음을 확인하세요:
- 파일 크기가 10MB 이하인지 확인
- 지원되는 파일 형식인지 확인
- 인터넷 연결 상태 확인

### 검색 및 필터링

**Q: 검색 결과가 나오지 않습니다.**
A: 다음을 확인하세요:
- 검색어 철자가 올바른지 확인
- 필터가 너무 제한적인지 확인
- 필터를 초기화하고 다시 시도

**Q: 필터 프리셋이 저장되지 않습니다.**
A: 브라우저의 로컬 스토리지가 활성화되어 있는지 확인하세요.

### 알림

**Q: 알림이 표시되지 않습니다.**
A: 다음을 확인하세요:
- 직원 정보에 자격증 만료일, 계약 종료일, 생일이 입력되어 있는지 확인
- 알림 목록 페이지에서 모든 알림 확인
- 브라우저 새로고침

**Q: 알림이 자동으로 업데이트되지 않습니다.**
A: 알림은 30초마다 자동 업데이트됩니다. 수동으로 새로고침하거나 알림 목록 페이지를 다시 열어보세요.

### 엑셀 연동

**Q: 엑셀 임포트가 실패합니다.**
A: 다음을 확인하세요:
- 템플릿 형식이 올바른지 확인
- 필수 항목이 모두 입력되었는지 확인
- 사원번호와 이메일이 중복되지 않는지 확인
- 임포트 결과 리포트에서 오류 상세 확인

**Q: 엑셀 파일이 다운로드되지 않습니다.**
A: 다음을 확인하세요:
- 브라우저의 다운로드 설정 확인
- 팝업 차단기가 활성화되어 있는지 확인
- 권한이 있는지 확인 (admin, hr 역할 필요)

### 파일 관리

**Q: 파일을 다운로드할 수 없습니다.**
A: 다음을 확인하세요:
- 파일이 존재하는지 확인
- 인터넷 연결 상태 확인
- 브라우저의 다운로드 설정 확인

**Q: 이력서 버전이 표시되지 않습니다.**
A: 이력서 타입으로 업로드한 파일만 버전 관리가 됩니다. 파일 타입을 확인하세요.

### 대시보드

**Q: 대시보드 차트가 표시되지 않습니다.**
A: 다음을 확인하세요:
- 직원 데이터가 있는지 확인
- 브라우저 콘솔에서 에러 확인
- 페이지 새로고침

**Q: 리포트 다운로드가 작동하지 않습니다.**
A: 브라우저의 다운로드 설정을 확인하고, 팝업 차단기가 비활성화되어 있는지 확인하세요.

### 기술적 문제

**Q: 페이지가 로드되지 않습니다.**
A: 다음을 확인하세요:
- 인터넷 연결 상태 확인
- 브라우저 캐시 삭제
- 다른 브라우저에서 시도
- 관리자에게 문의

**Q: 데이터가 저장되지 않습니다.**
A: 다음을 확인하세요:
- 필수 항목이 모두 입력되었는지 확인
- 중복된 사원번호나 이메일이 아닌지 확인
- 브라우저 콘솔에서 에러 확인
- 페이지 새로고침 후 다시 시도

## 추가 도움말

문제가 해결되지 않으면:
1. 브라우저 개발자 도구 (F12)에서 에러 확인
2. 관리자에게 문의
3. [빠른 시작 가이드](./QUICK_START.md) 참고
4. [문제 해결 가이드](./QUICK_START.md#-문제-해결) 참고
