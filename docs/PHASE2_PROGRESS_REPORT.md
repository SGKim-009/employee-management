1. 브라우저 개발자 도구에서 모바일 뷰 활성화
2. ✅ 네비게이션 바에 햄버거 메뉴 표시 확인
3. ✅ 직원 카드가 1열로 표시되는지 확인
4. ✅ 페이지네이션이 세로 레이아웃으로 표시되는지 확인
5. ✅ 모든 버튼이 최소 44x44px 크기인지 확인

#### 태블릿 뷰 (768px ~ 1023px)
1. ✅ 직원 카드가 2열로 표시되는지 확인

#### 데스크톱 뷰 (1024px 이상)
1. ✅ 직원 카드가 3열로 표시되는지 확인

### 다크 모드 테스트

1. 네비게이션 바의 테마 전환 버튼 클릭
2. ✅ 다크 모드로 전환되는지 확인
3. ✅ 모든 컴포넌트가 다크 모드 스타일로 표시되는지 확인
4. ✅ 페이지 새로고침 후에도 다크 모드가 유지되는지 확인

1. ✅ 직원 카드가 순차적으로 등장하는지 확인

3. ✅ 모달이 애니메이션과 함께 나타나는지 확인

3. ✅ 스킵 링크가 작동하는지 확인
