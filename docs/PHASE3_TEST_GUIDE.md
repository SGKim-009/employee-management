## 대시보드 및 분석 테스트

### 테스트 항목: 통계 데이터 정확성 확인

#### 1. 전체 직원 수 통계

**절차:**
1. 대시보드 페이지(`/dashboard`) 접속
2. "전체 직원" 카드 확인
3. 메인 페이지의 실제 직원 수와 비교

**예상 결과:**
- 대시보드의 전체 직원 수가 메인 페이지의 직원 수와 일치
- 재직/휴직/퇴사 상태별 합계가 전체 직원 수와 일치

#### 2. 부서별 인원 분포

**절차:**
1. 대시보드에서 "부서별 분포" 섹션 확인
2. 각 부서별 인원 수 확인
3. 메인 페이지에서 필터로 각 부서별 직원 수 확인

**예상 결과:**
- 대시보드의 부서별 인원 수가 실제 필터링 결과와 일치
- 모든 부서의 합계가 전체 직원 수와 일치

#### 3. 직급별 분포

**절차:**
1. 대시보드에서 "직급별 분포" 섹션 확인
2. 각 직급별 인원 수 확인
3. 메인 페이지에서 필터로 각 직급별 직원 수 확인

**예상 결과:**
- 대시보드의 직급별 인원 수가 실제 필터링 결과와 일치
- 모든 직급의 합계가 전체 직원 수와 일치

#### 4. 입사/퇴사 추이 (최근 30일)

**절차:**
1. 대시보드에서 "최근 30일 입사/퇴사" 통계 확인
2. 실제 데이터와 비교:
   - 입사일이 최근 30일 이내인 직원 수
   - 퇴사일이 최근 30일 이내인 직원 수

**예상 결과:**
- 통계 수치가 실제 데이터와 일치
- 날짜 계산이 정확함 (30일 기준)

---

### 테스트 항목: 차트 데이터 정확성 확인

#### 1. 부서별 인원 파이 차트

**절차:**
1. 대시보드에서 파이 차트 확인
2. 각 부서별 비율 확인
3. 실제 데이터와 비교

**예상 결과:**
- 파이 차트의 비율이 실제 부서별 인원 비율과 일치
- 차트가 정상적으로 렌더링됨
- 호버 시 정확한 수치 표시

#### 2. 입사 추이 라인 차트

**절차:**
1. 대시보드에서 라인 차트 확인
2. 최근 6개월 또는 12개월 입사 추이 확인
3. 실제 입사 데이터와 비교

**예상 결과:**
- 라인 차트의 데이터 포인트가 실제 입사 데이터와 일치
- 차트가 정상적으로 렌더링됨
- 시간축이 정확함

#### 3. 급여 분포 히스토그램

**절차:**
1. 대시보드에서 히스토그램 확인
2. 급여 구간별 분포 확인
3. 실제 급여 데이터와 비교

**예상 결과:**
- 히스토그램의 구간별 분포가 실제 급여 분포와 일치
- 차트가 정상적으로 렌더링됨

### 테스트 항목: 리포트 생성 및 다운로드 확인

#### 1. CSV 리포트 다운로드

**절차:**
1. 대시보드에서 "리포트 다운로드" 버튼 클릭
2. "CSV" 옵션 선택
3. 다운로드된 파일 확인

**예상 결과:**
- CSV 파일이 정상적으로 다운로드됨
- 파일명에 날짜가 포함됨 (예: `employee-report-2024-01-15.csv`)
- CSV 파일을 엑셀에서 열었을 때 한글이 깨지지 않음
- 모든 직원 데이터가 포함됨
- 헤더 행이 정확함

#### 2. JSON 리포트 다운로드

**절차:**
1. 대시보드에서 "리포트 다운로드" 버튼 클릭
2. "JSON" 옵션 선택
3. 다운로드된 파일 확인

**예상 결과:**
- JSON 파일이 정상적으로 다운로드됨
- 파일명에 날짜가 포함됨
- JSON 형식이 유효함 (JSON 파서로 검증 가능)
- 모든 직원 데이터가 포함됨
- 리포트 메타데이터(생성일시, 요약 통계) 포함

#### 3. 텍스트 리포트 다운로드

**절차:**
1. 대시보드에서 "리포트 다운로드" 버튼 클릭
2. "텍스트" 옵션 선택
3. 다운로드된 파일 확인

**예상 결과:**
- 텍스트 파일이 정상적으로 다운로드됨
- 파일명에 날짜가 포함됨
- 리포트 헤더, 요약 통계, 직원 목록이 포함됨
- 텍스트 포맷이 읽기 쉬움

## 알림 시스템 테스트

### 테스트 항목: 만료 임박 자격증 알림 확인

#### 1. 자격증 만료 30일 전 알림

**절차:**
1. 직원 정보에 자격증 추가:
   - 자격증명: "정보처리기사"
   - 만료일: 오늘부터 30일 후
2. 알림 페이지(`/notifications`) 접속
3. 알림 목록 확인

**예상 결과:**
- "자격증 만료 임박" 알림이 표시됨
- 알림 메시지에 자격증명과 만료일이 포함됨
- 알림 우선순위가 "medium" 또는 "high"
- 알림 배지에 숫자가 표시됨

#### 2. 자격증 만료 7일 전 알림

**절차:**
1. 직원 정보에 자격증 추가:
   - 만료일: 오늘부터 7일 후
2. 알림 페이지 확인

**예상 결과:**
- "자격증 만료 임박" 알림이 표시됨
- 알림 우선순위가 "high" 또는 "urgent"
- 알림 메시지에 긴급성 강조

#### 3. 자격증 만료 알림

**절차:**
1. 직원 정보에 자격증 추가:
   - 만료일: 오늘 또는 과거 날짜
2. 알림 페이지 확인

**예상 결과:**
- "자격증 만료" 알림이 표시됨
- 알림 우선순위가 "urgent"
- 알림 메시지에 만료 상태 명시

## 파일 관리 테스트

### 테스트 항목: 파일 업로드/다운로드 확인

#### 1. 문서 업로드

**절차:**
1. 직원 상세 페이지에서 "파일 관리" 링크 클릭
2. "파일 업로드" 버튼 클릭
3. 문서 파일 선택 (PDF, DOCX 등)
4. 파일 타입 선택: "문서"
5. 설명 입력 (선택사항)
6. 업로드 확인

**예상 결과:**
- 파일이 정상적으로 업로드됨
- 파일 목록에 새 파일이 표시됨
- 파일명, 크기, 업로드 날짜가 정확히 표시됨
- Toast 메시지로 업로드 성공 알림

#### 2. 파일 다운로드

**절차:**
1. 파일 목록에서 다운로드할 파일 클릭
2. 다운로드 확인

**예상 결과:**
- 파일이 정상적으로 다운로드됨
- 다운로드된 파일이 원본과 동일함
- 파일명이 정확함

#### 3. 파일 삭제

**절차:**
1. 파일 목록에서 삭제할 파일의 삭제 버튼 클릭
2. 확인 대화상자에서 확인
3. 파일 목록 확인

**예상 결과:**
- 파일이 목록에서 제거됨
- Toast 메시지로 삭제 성공 알림
- Storage에서도 파일이 삭제됨

---

### 테스트 항목: 이력서 관리 기능 확인

#### 1. 이력서 업로드 및 버전 관리

**절차:**
1. 직원의 파일 관리 페이지 접속
2. 이력서 파일 업로드 (파일 타입: "이력서")
3. 동일한 직원의 새 이력서 업로드
4. 버전 번호 확인

**예상 결과:**
- 첫 번째 이력서: 버전 1, 최신 버전 표시
- 두 번째 이력서: 버전 2, 최신 버전 표시
- 첫 번째 이력서의 "최신 버전" 표시가 제거됨
- 버전 번호가 자동으로 증가함

#### 2. 최신 버전으로 설정

**절차:**
1. 이전 버전의 이력서 선택
2. "최신 버전으로 설정" 버튼 클릭
3. 버전 상태 확인

**예상 결과:**
- 선택한 이력서가 최신 버전으로 설정됨
- 다른 버전들의 "최신 버전" 표시가 제거됨
- Toast 메시지로 설정 성공 알림

---

### 테스트 항목: 계약서 관리 기능 확인

#### 1. 계약서 업로드 및 만료일 설정

**절차:**
1. 직원의 파일 관리 페이지 접속
2. 계약서 파일 업로드 (파일 타입: "계약서")
3. 만료일 입력 (예: 2024-12-31)
4. 업로드 확인

**예상 결과:**
- 계약서가 정상적으로 업로드됨
- 만료일이 파일 정보에 표시됨
- 만료일이 임박하면 알림 생성됨

#### 2. 계약서 만료일 수정

**절차:**
1. 계약서 파일 선택
2. "만료일 수정" 버튼 클릭
3. 새 만료일 입력
4. 저장 확인

**예상 결과:**
- 만료일이 정상적으로 업데이트됨
- 파일 정보에 새 만료일이 표시됨
- Toast 메시지로 수정 성공 알림

## 엑셀 연동 테스트

### 테스트 항목: 엑셀 파일 임포트 확인

#### 1. 엑셀 템플릿 다운로드

**절차:**
1. 메인 페이지에서 "엑셀 템플릿 다운로드" 버튼 클릭
2. 다운로드된 파일 확인

**예상 결과:**
- 엑셀 템플릿 파일이 다운로드됨
- 템플릿에 모든 필수 필드가 포함됨
- 헤더 행이 한글로 표시됨
- 예시 데이터가 포함됨

#### 2. 엑셀 파일 임포트

**절차:**
1. 메인 페이지에서 "엑셀 임포트" 버튼 클릭
2. 준비된 엑셀 파일 선택
3. 임포트 진행
4. 임포트 결과 리포트 확인

**예상 결과:**
- 엑셀 파일이 정상적으로 파싱됨
- 데이터 검증이 수행됨
- 성공/실패한 행 수가 표시됨
- 오류가 있는 행은 오류 메시지와 함께 표시됨
- 성공한 직원이 목록에 추가됨

#### 3. 중복 데이터 처리

**절차:**
1. 이미 존재하는 사원번호를 포함한 엑셀 파일 임포트
2. 중복 처리 옵션 확인

**예상 결과:**
- 중복 사원번호가 감지됨
- 중복 처리 옵션이 제공됨 (건너뛰기, 업데이트)
- 적절한 오류 메시지 표시

---

### 테스트 항목: 엑셀 파일 다운로드 및 형식 확인

#### 1. 직원 목록 엑셀 다운로드

**절차:**
1. 메인 페이지에서 "엑셀 다운로드" 버튼 클릭
2. 다운로드된 파일 확인

**예상 결과:**
- 엑셀 파일이 정상적으로 다운로드됨
- 파일명에 날짜가 포함됨
- 모든 직원 데이터가 포함됨
- 현재 필터/정렬 상태가 반영됨 (선택사항)

#### 2. 엑셀 파일 형식 확인

**절차:**
1. 다운로드된 엑셀 파일을 엑셀에서 열기
2. 데이터 형식 확인

**예상 결과:**
- 한글이 정상적으로 표시됨
- 날짜 형식이 올바름
- 숫자 형식이 올바름 (급여 등)
- 모든 컬럼이 정확히 매핑됨
- 헤더 행이 명확함

#### 3. 엑셀 파일 데이터 정확성

**절차:**
1. 다운로드된 엑셀 파일의 데이터 확인
2. 웹 페이지의 직원 정보와 비교
