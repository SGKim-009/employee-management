## 🔧 기술 스택

### 프론트엔드
- **Framework**: Next.js 16.1.1 (App Router)
- **언어**: TypeScript
- **스타일링**: Tailwind CSS 4
- **상태 관리**: React Hooks (useState, useCallback, useMemo)
- **폼 관리**: React Hook Form + Zod
- **차트**: Recharts
- **엑셀**: xlsx
- **아이콘**: Lucide React

### 백엔드
- **BaaS**: Supabase
- **데이터베이스**: PostgreSQL (Supabase)
- **인증**: Supabase Auth
- **Storage**: Supabase Storage

### 배포
- **플랫폼**: Vercel (권장)
- **환경 변수**: `.env.local`

## 🚀 주요 기능 상세

### 1. 직원 관리
- 직원 등록/수정/삭제
- 직원 상세 정보 조회
- 프로필 이미지 업로드
- 인사관리카드 출력
- 고급 검색 및 필터링
- 정렬 기능
- 무한 스크롤

### 2. 대시보드
- 전체 직원 수 통계
- 부서별/직급별 분포 차트
- 입사/퇴사 추이 그래프
- 급여 분포 히스토그램
- 리포트 다운로드 (CSV, JSON, 텍스트)

### 3. 알림 시스템
- 자격증 만료 알림 (30일 전, 7일 전)
- 계약 갱신 알림 (30일 전, 7일 전, 만료)
- 생일 알림 (오늘, 7일 후)
- 알림 읽음/삭제 관리
- 실시간 알림 배지

### 4. 파일 관리
- 문서 업로드/다운로드
- 이력서 버전 관리 (자동 버전 번호, 최신 버전 표시)
- 계약서 만료일 관리
- 파일 타입별 필터링 (문서, 이력서, 계약서, 기타)

### 5. 엑셀 연동
- 엑셀 파일 임포트 (일괄 직원 등록)
- 엑셀 템플릿 다운로드
- 임포트 결과 리포트 (성공/실패 통계, 오류 상세)
- 직원 목록 엑셀 다운로드
